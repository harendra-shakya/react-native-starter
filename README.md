# A React Native Tailwind Firebase Ts Start By Harendra

Start expo server

```bash
yarn start
```

Use this if above command doesn't work:

```bash
yarn start --tunnel
```

Launch the development apps

```bash
# start android app
yarn android

# start ios app
yarn ios

# start web app
yarn web
```

Use _Tailwind-rn_

```bash
# start Tailwind class generator (in watch mode)
yarn dev:tailwind

# build Tailwind classes
yarn build:tailwind
```
